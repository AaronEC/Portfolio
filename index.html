IBCalm AI Assistant

A purpose built, highly scalable AI assistant designed to help IBS sufferers manage and reduce their symptoms. Running a HTML/JS frontend with a python backend linked through the Flask API, utlising Azure TTS for accessibility, the assistant is packaged with Docker making for easy deployment and scalability, currently deployed via AWS Lightsail.

Burnham Company of Archers Club Website (WIP)

An informational club website created for my local archery club featuring live news feed and online archery range booking system for club members (powered by Azure). Created with Gatsby for React, to ensure that the site can be deployed as a static website whilst also featuring a live newsfeed which doesn't require direct editing of the HTML files, to ensure ease of use for the clubs committee members.

Table Tennis League Organiser

A comprehensive standalone app for assisting in organising and running a table tennis league. Deployed for use in a real league with over 200 players, the application, developed in Java with a JavaFX front end, has an extensive feature list including; Multiple user support, full database integration, live saving, automated league fixture creation, score and points calculation.

IBCalm WebApp (WIP)

A fully accessible web application, with a HTML/JavaScript front end. This app features a Python AI integrated with Flask, designed to help alleviate the users IBS symptoms. This is achieved using an expert case-based reasoning system assesing users on their dietary intake and subsequent symptoms, linked to the IBC Ai assistant. Features complete AI backend with SQL database implementation.
